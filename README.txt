Projekt końcowy klasyfikacja
Projekt wykonany został w trakcie kursu Data Scientist prowadzonego przez firmę Software Development Academy na zlecenie Pomorskiej Fundacji Inicjatyw Gospodarczych.
Jest to projekt kończący moduł "Machine Learning w praktyce: problemy klasyfikacji w uczeniu nadzorowanym"
​
Wprowadzenie
W tym projekcie zostanie przedstawiony algorytm, który ma na celu jak przewidzenie satysfakcji klienta po odbytym locie
​
Instalacja
Sklonuj repozytorium: git clone https://github.com/KamilShape/passengers_satistfaction.git
Przejdź do folderu z projektem: cd passengers_satistfaction
